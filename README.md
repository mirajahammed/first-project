1. LabReport01-Inheritance
a. Short Description:
   This project is created in Java. It demonstrates inheritance between Employee and Manager classes.
   The displayInfo() method shows all employee details along with the department information.
b. Features:
   Employee and Manager class.
   Employee details like name, ID, and salary.
   Manager’s department along with employee details.
   Inheritance in Java.
   Method overriding with displayInfo().
c.  Sample output:
    Employee Details
  Employee Name: Akbar
  Employee ID: 242002110
  Basic Salary: 20000.0

    Manager Details
  Employee Name: Anik
  Employee ID: 242002100
  Basic Salary: 40000.0
  Department: Sales
